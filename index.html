Learning Verilog with YoSys

Verilog

This blog series attempts to provide a starting point to learning Verilog.

YoSys

YoSys is a new logic systhesis tool from Clifford Wolf. It is completely open-source, and perfect for learning Verilog with.

Xilinx

For actual implementation we shall synthesise our YoSys netlists using the Xilinx Vivado suite, with a web-pack license.
